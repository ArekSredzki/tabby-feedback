## About Tabby

Tabby is a Chrome extension for intelligent tab management featuring:

- AI chat assistant for managing tabs via natural language
- Smart tab grouping suggestions
- Duplicate tab detection
- Session recovery with timeline visualization
- Memory usage monitoring and optimization
- Tab group persistence via bookmarks
